# Hotel_Booking_Confirmation_Page
A simulation for a hotel booking confirmation page. The website is well-organized into five main categories: Left sidebar, Dashboard  top, visitior stary receipt, welcome card, and bottom info cards.
## Lift Sidebar
A sidebar including the hotel brand, and menu bar. The menu bar includes more details about the apartment, town, stay time , and messages for communication with the host. 
## Dashboard Top
It ensures the booking confirmation. It includes two buttons for printing the hotel receipt and add stay timetable to visitor's calendar.
## Stay Receipt 
Includes the financial information about the staying cost, and the way of paying.
## Welcome Card
Just a welcome card from the host.
## Bottom Info Cards
3 cards includes more information that facilitate on visitors during their stay time.

The two buttons in the top right are designed with hover state.
